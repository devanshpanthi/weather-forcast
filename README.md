# Weather Forecast

A responsive weather forecast website built with HTML, CSS, vanilla JavaScript, and the free Open-Meteo APIs.

## Run

Open `index.html` in a modern browser, or start it with Live Server. An internet connection is required to retrieve city and weather data.

## Features

- Indian city, district, and local block search through OpenStreetMap Nominatim
- Current conditions, hourly forecast, and seven-day forecast
- Responsive dark blue glassmorphism interface
- SVG weather icons, loading feedback, and error handling
